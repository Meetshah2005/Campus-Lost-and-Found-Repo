# Campus Lost and Found

Campus Lost & Found System A web-based Lost and Found application designed to help university students quickly report, search, and recover lost items. The platform streamlines the recovery process by enabling users to submit detailed reports, upload images, view campus map locations, and receive automated match notifications. 

Design Goals: 
Efficiency of Recovery — no more than 2 interactions to initiate item return 
Ease of Navigation — complete key tasks within 4 steps 
Data Security — user anonymity preserved until recovery is confirmed
